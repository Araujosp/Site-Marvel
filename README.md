# 🦸‍♂️ Site Marvel - Heróis e Vilões

Este projeto consiste em um site desenvolvido com **HTML e CSS** inspirado no universo da **Marvel**, apresentando os **heróis favoritos do público** e uma página dedicada aos **vilões mais populares**.

O site foi desenvolvido como parte dos estudos em **Desenvolvimento de Sistemas pelo SENAI**, com foco em **Front-End e criação de páginas web**.

---

## 📌 Funcionalidades

O site possui:

- Página inicial com ranking de heróis
- Página de vilões favoritos
- Página de login
- Página "Fale Conosco"
- Página "Sobre Nós"
- Menu de navegação
- Cartões com imagens dos personagens
- Descrição dos personagens
- Botões "Saiba mais"
- Layout moderno
- Fundo estilizado

---

## 📂 Páginas do Site

### 🏠 Página Inicial
Apresenta o ranking dos heróis favoritos do público da Marvel com imagens e descrições.

### 🦹 Página de Vilões
Mostra alguns dos vilões mais populares da Marvel com imagens e informações.

### 🔐 Página de Login
Permite que o usuário simule um acesso ao sistema através de um formulário simples.

### 📩 Página Fale Conosco
Possui um formulário onde o usuário pode enviar mensagens ou dúvidas.

### 👥 Página Sobre Nós
Apresenta informações sobre o projeto e o desenvolvedor.
